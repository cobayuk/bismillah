Contoh Soal dan Pembahasan TIU SKD bagian 10

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. NELAYAN : PERAHU = ... : ...
    1. Penulis : buku
    2. Petani : traktor
    3. Pengacara : hukum
    4. Guru : murid
    5. Berpikir : otak
    Jawaban : B
    Nelayan : perahu = petani : traktor. Nelayan bekerja dengan menggunakan perahu, sedangkan petani bekerja dengan menggunakan traktor.
  2. PANGGUNG : AKTOR = ... : ...
    1. Perpustakaan : siswa
    2. Keamanan : satpam
    3. Ring : petinju
    4. Petani : cangkul
    5. Rumah sakit : dokter
    Jawaban : C
    Panggung : aktor = ring : petinju. Panggung merupakan tempat aktor beraksi, ring merupakan tempat petinju beraksi.
  3. POP : MUSIK = ... : ...
    1. Farmakologi : ilmu
    2. Film : skenario
    3. Drama : panggung
    4. Sandiwara : plot
    5. Teater : acara
    Jawaban : C
    Pop : musik = farmalogi : ilmu. Keduanya sama - sama mempunyai hubungan "salah satu jenis".
  4. RAMALAN : ASTROLOGI = ... : ...
    1. Ilmu : biologi
    2. Perusahaan : akutansi
    3. Cita - cita : belajar
    4. Bangsa : etnologi
    5. Negara : presiden
    Jawaban : D
    Ramalan : astrologi = bangsa : etnologi. Keduanya sama - sama mempunyai hubungan tentang "ilmu yang mempelajari".
  5. UGAHARI = .....
    1. Canggih
    2. Bergantung
    3. Aturan
    4. Kapal laut
    5. Sederhana
    Jawaban : E
    Ugahari artinya sedang, pertengahan, atau sederhana.
  6. VANDALISME = ....
    1. Perusakan
    2. Gerakan perubahan
    3. Penambahan
    4. Ambisi
    5. Penurunan
    Jawaban : A
    Vandalisme artinya penghancuran atau perusakan.
  7. BALAI YASA = ....
    1. Rumah sakit
    2. Rumah makan
    3. Aula
    4. Ruang pertemuan
    5. Bengkel lokomotif
    Jawaban : E
    Balai Yasa = bengkel lokomotif.
  8. RAMAI >< ....
    1. Sepi
    2. Sorak
    3. Lengang
    4. Penuh
    5. Sesak
    Jawaban : A
    Lawan kata yang tepat untuk kata ramai adalah kata sepi.
  9. MONOGAMI >< ...
    1. Tikus
    2. Poligami
    3. Sejenis
    4. Wanita
    5. Cergam
    Jawaban : B
    Lawan kata yang tepat untuk kata monogami adalah kata poligami.
  10. ABSURD >< ....
    1. Praktis
    2. Kacau
    3. Runtuh
    4. Rasional
    5. Tegap
    Jawaban : D
    Absurd = Tidak masuk akal. Lawan kata yang tepat untuk kata absurd adalah kata rasional yang berarti masuk akal.
