Contoh Soal dan Pembahasan TIU SKD bagian 12

Jun 11, 2020

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. MATAHARI : TERIK = ... : ...
    1. Lampu : bersinar
    2. Bumi : bulan
    3. Merkurius : bumi
    4. Panas : api
    5. Bulan : benderang
    Jawaban : B
    Matahari : terik = bulan : benderang. Keduanya sama - sama mempunyai hubungan tentang "bersinar dengan". Matahari bersinar dengan terik , sedangkan bulan bersinar dengan benderang.
  2. PENULIS : BUKU = ... : ...
    1. Masinis : kereta api
    2. Sopir : bus
    3. Komponis : lagu
    4. Rumah : atap
    5. Konsumen : produsen
    Jawaban : C
    Penulis : buku = komponis : lagu. Keduanya sama - sama mempunyai hubungan tentang "menghasilkan karya".
  3. TELLER : BANK = ... : RESTORAN
    1. Masakan
    2. Pelayan
    3. Minuman
    4. Bartender
    5. Menu
    Jawaban : B
    Teller : bank = pelayan : restoran. Keduanya sama - sama mempunyai hubungan tentang "bekerja di".
  4. ULAT : SUTRA = LENAH : ...
    1. Kepompong
    2. Benang
    3. Bunga
    4. Madu
    5. Pohon
    Jawaban : D
    Ulat : sutra = lebah : madu. Keduanya sama - sama mempunyai hubungan tentang "menghasilkan".
  5. FATSUN = .....
    1. Kepala
    2. Etika
    3. Indisiplin
    4. Perulangan
    5. Budaya
    Jawaban : B
    Fatsun = sopan santun, etika, etiket.
  6. PANDIR = ....
    1. Agak pintar
    2. Bodoh
    3. Pandai hadir
    4. Tidak jenius
    5. Pemandangan
    Jawaban : B
    Pandir artinya tidak berilmu, bodoh, bebal.
  7. ITERASI = ....
    1. Interaksi
    2. Hubungan
    3. Perulangan
    4. Korelasi
    5. Kedekatan
    Jawaban : C
    Iterasi artinya adalah perulangan.
  8. OUTPUT >< ....
    1. Pemasukan
    2. Pengeluaran
    3. Pelaksanaan
    4. Rencana
    5. Tujuan
    Jawaban : A
    Output = Pengeluaran. Lawan kata yang tepat untuk kata output adalah kata pemasukan.
  9. FIKSI >< ....
    1. Imajinasi
    2. Maya
    3. Fana
    4. Khayalan
    5. Nyata
    Jawaban : E
    Fiksi = Tidak berdasarkan kenyataan. Lawan kata yang tepat untuk kata fiksi adalah kata nyata.
  10. EPILOG >< ....
    1. Hipolog
    2. Dialog
    3. Monolog
    4. Analog
    5. Prolog
    Jawaban : E
    Epilog = Penutup. Lawan kata yang tepat untuk kata epilog adalah kata Prolog.
