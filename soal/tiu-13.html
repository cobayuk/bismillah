Contoh Soal dan Pembahasan TIU SKD bagian 13

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. HAUS : MINUM = LELAH : ...
    1. Berlari
    2. Istirahat
    3. Santai
    4. Makan
    5. Berbaring
    Jawaban : B
    Jika merasa haus harus minum untuk menghilangkannya. Jika merasa lelah harus beristirahat untuk menghilangkannya.
  2. SEPEDA : KAYUH = ... : ...
    1. Terminal : Bis
    2. Motor : Balapan
    3. Perahu : Dayung
    4. Kail : Nelayan
    5. Pasir : Semen
    Jawaban : C
    Sepeda harus dikayuh agar dapat bergerak. Perahu harus didayung agar dapat bergerak.
  3. UANG : BANK = PADI : ...
    1. Sawah
    2. Lumbung
    3. Jambangan
    4. Kapas
    5. Tempayan
    Jawaban : B
    Uang disimpan di bank. Padi disimpan dilumbung.
  4. DELMAN : KUDA = HELIKIOPTER : ...
    1. Tiket
    2. Ban
    3. Baling - baling
    4. Landasan
    5. Penumpang
    Jawaban : C
    Delman digerakkan oleh kuda. Helikopter digerakkan oleh baling - baling.
  5. MUKADIMAH = ....
    1. Halaman
    2. Pengantar
    3. Pembukaan
    4. Isi
    5. Penutup
    Jawaban : B
    Mukadimah artinya pendahuluan atau kata pengantar.
  6. PROKSIMAL = .....
    1. Terdekat
    2. Distal
    3. Terjauh
    4. Maksimal
    5. Pendekatan
    Jawaban : A
    Proksimal = dekat ke pusat; terletak pada sarah pusat suatu benda. Proksimal = terdekat.
  7. ARTIFISIAL = .....
    1. Alami
    2. Murni
    3. Pabrikan
    4. Campuran
    5. Buatan
    Jawaban : E
    Artifisial artinya tidak alami, atau buatan.
  8. EKLEKTIK >< ....
    1. Selektif
    2. Pilih - pilih
    3. Tidak pilih - pilih
    4. Lentik
    5. Akurat
    Jawaban : C
    Ekletik = Memilih yang terbaik. Lawan kata yang tepat untuk kata elektik adalah kata tidak pilih - pilih.
  9. SINKRON >< .....
    1. Selaras
    2. Serasi
    3. Sumbang
    4. Harmonis
    5. Sesuai
    Jawaban : C
    Sinkron = Sesuai; Selaras. Lawan kata yang tepat untuk kata sinkron adalah kata sumbang.
  10. SEKULER >< .....
    1. Keagamaan
    2. Duniawi
    3. Rohaniah
    4. Cendikia
    5. Tradisional
    Jawaban : A
    Sekuler = Bersifat duniawi atau kebendaan (bukan keagamaan). Lawan kata yang tepat untuk kata sekuler adalah kata keagamaan.
