Contoh Soal dan Pembahasan TIU SKD bagian 14

Jun 11, 2020

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. BASKET : RING = ... : ...
    1. Badminton : Kok
    2. Sepak bola : Gawang
    3. Renang : Kolam
    4. Sepak bola : Lapangan
    5. Tinju : Sarung tangan
    Jawaban : B
    Dalam permainan basket, bola dimasukkan ke dalam ring. Dalam permainan sepakbola, bola dimasukkan ke dalam gawang.
  2. SENIN : RABU = ... : ...
    1. Satu : Tiga
    2. Mawar : Melati
    3. Detik : Menit
    4. Padi : Beras
    5. April : Februari
    Jawaban : A
    Dua hari setelah hari sening adalah hari rabu. Dua angka setelah angka satu adalah angka tiga.
  3. JANUARI : DESEMBER = ... : ...
    1. Maret : April
    2. Kemarin : Lusa
    3. Pembuka : Penutup
    4. Selasa : Jumat
    5. Satu : Sepuluh
    Jawaban : C
    Bulan Januari adalah bulan pembuka, dan bulan Desember adalah bulan penutup.
  4. TERANG : MATAHARI = PANAS : ...
    1. Dingin
    2. Segar
    3. Api
    4. Lampu
    5. Bintang
    Jawaban : C
    Yang bersinar terang adalah matahari. Yang terasa panas adalah api.
  5. NANAR = ....
    1. Kosong
    2. Luka
    3. Tajam
    4. Berani
    5. Bingung
    Jawaban : E
    Nanar = berasa pusing (karena mabuk, kena pukul, dsb); agak hilang akal (karena kelelahan, dsb), bingung; tercengang - cengang.
  6. TALENTA = ....
    1. Cita - cita
    2. Kegemaran
    3. Bakat
    4. Keinginan
    5. Hobi
    Jawaban : C
    Talenta = pembawaan seseorang sejak lahir, atau bakat.
  7. AMBIVALEN = ....
    1. Setia
    2. Kokoh
    3. Bercabang
    4. Ibarat
    5. Kahar
    Jawaban : C
    Ambivalen artinya bercabang, mendua, terpecah.
  8. AKTUAL >< ....
    1. Modern
    2. Fiktif
    3. Nyata
    4. Baru
    5. Sebenarnya
    Jawaban : B
    Aktual = Berdasarkan kenyataan. Lawan kata yang tepat untuk kata aktual adalah kata fiktif.
  9. ABOLISI >< ....
    1. Pemberatan
    2. Keringanan
    3. Pengurangan
    4. Pemotongan
    5. Pangampunan
    Jawaban : A
    Abolisi = Pembatalan (peringanan) hukuman. Lawan kata yang tepat untuk kata abolisi adalah kata pemberatan.
  10. HARMONI >< ...
    1. Setara
    2. Sesuai
    3. Selaras
    4. Cocok
    5. Sumbang
    Jawaban : E
    Harmoni = Selar as. Lawan kata yang tepat untuk kata harmoni adalah kata sumbang.
