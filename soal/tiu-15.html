Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. MAKANAN : LAPAR : KENYANG = ... : ... : ...
    1. Bulan : Bintang : Malam
    2. Lampu : Gelap : Terang
    3. Siang : Terang : Matahari
    4. Gelap : Terang : Matahari
    5. Air : Dahaga : Haus
    Jawaban : B
    Makanan diperlukan saat lapar, agar menjadi kenyang. Lampu diperlukan saat gelap, agar menjadi terang.
  2. HURUF : KATA : CERITA = ... : ... : ...
    1. Luar angkasa : Astronomi : Fosil
    2. Padi : Karung : Lumbung
    3. Jarang : Sering : Galaksi
    4. Bintang : Galaksi : Alam semesta
    5. Saya : Kita : Mereka
    Jawaban : D
    Rangkaian huruf membentuk kata, dan rangkaian kata membentuk cerita. Kumpulan bintang membentuk galaksi, dan rangkaian galaksi membentuk alam semesta.
  3. APEL : POHON = ... : ...
    1. Kayu : Triplek
    2. Batako : Tembok
    3. Pohon : Kayu
    4. Emas : Tambang
    5. Ayam : Kandang
    Jawaban : D
    Apel diambil dari pohon. Emas diambil dari tambang.
  4. HUJAN : BASAH = BAKAR : ...
    1. Hangus
    2. Mati
    3. Habis
    4. Keringat
    5. Dingin
    Jawaban : A
    Jika terkena hujan akan menjadi basah. Jika terbakar akan menjadi hangus.
  5. HIBRIDASI = ...
    1. Penyilangan
    2. Penjaringan
    3. Unggulan
    4. Penyuburan
    5. Penghijauan
    Jawaban : A
    Hibridasi = persilangan dari populasi yang berbeda.
  6. SUPREMASI = ...
    1. Melebihi jangkauan
    2. Bersifat komplek
    3. Kekuasaan tertinggi
    4. Hal pengangkatan
    5. Tambahan
    Jawaban : C
    Supremasi = kekuasaan tertinggi (teratas).
  7. KONTEMPORER = ...
    1. Aneh
    2. Kuno
    3. Pada masa kini
    4. Abstrak
    5. Tidak beraturan
    Jawaban : C
    Kontemporer = pada waktu yang sama; semasa; pada masa kini; dewasa ini.
  8. GASAL >< ...
    1. Ganjil
    2. Gangsal
    3. Bangsal
    4. Genap
    5. Gatal
    Jawaban : D
    Gasal = Ganjil. Lawan kata yang tepat untuk kata ganjil adalah kata genap.
  9. PEJAL >< ....
    1. Padat
    2. Padu
    3. Berongga
    4. Kimpal
    5. Masif
    Jawaban : C
    Pejal = Padat; Penuh. Lawan kata yang tepat untuk kata pejal adalah kata berongga.
  10. PASCA >< ....
    1. Awal
    2. Mulai
    3. Sekarang
    4. Sebelum
    5. Sesudah
    Jawaban : D
    Pasca = Sesudah. Lawan kata yang tepat untuk kata pasca adalah kata sebelum.
