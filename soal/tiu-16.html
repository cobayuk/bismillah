Contoh Soal dan Pembahasan TIU SKD bagian 16

Jun 11, 2020

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. PERANGKAP : MENANGKAP = ... : ...
    1. Kapal : Berlabuh
    2. Kereta : Langsing
    3. Pancing : Ikan
    4. Senapan : Berburu
    5. Parang : Pengasah
    Jawaban : D
    Perangkap digunakan untuk menangkap. Senapan digunakan untuk berburu.
  2. DIAGONAL : SEGI EMPAT = ... : ...
    1. Diameter : Lingkaran
    2. Tinggi : Segitiga
    3. Radius : Busur
    4. Luas : Panjang
    5. Sudut : Tegak lurus
    Jawaban : A
    Garis diagonal membagi segi empat menjadi dua. Garis diameter membagi lingkaran menjadi dua.
  3. ROKOK : ASBAK = AIR : ...
    1. Ember
    2. Pancur
    3. Selokan
    4. Angin
    5. Keran
    Jawaban : A
    Tempat untuk menampung rokok adalah asbak. Tempat untuk menampung air adalah ember.
  4. ITALIA : 2006 = ... : ...
    1. Prancis : 2006
    2. Italia : 1998
    3. Brazil : 1994
    4. Jerman Barat : 1998
    5. Argentina : 1990
    Jawaban : C
    Italia adalah juara Piala Dunia sepakbola tahun 2006. Brazil adalah juara Piala Dunia sepakbola tahun 1994.
  5. PURBAWISESA = ...
    1. Selesai
    2. Acara
    3. Bergelombang
    4. Hubungan
    5. Kekuasaan sepenuh - penuhnya
    Jawaban : E
    Purbawisesa = kekuasaan sepenuh - penuhnya.
  6. PARAS = ...
    1. Ganti
    2. Cukur
    3. Wajah
    4. Salah
    5. Payah
    Jawaban : C
    Paras = rupa muka; wajah.
  7. ABRASI = ...
    1. Erosi
    2. Kelupas
    3. Kikis
    4. Keduk
    5. Hancur
    Jawaban : C
    Abrasi = pengikisan batuan oleh air, es atau angin yang mengandung dan mengangkut hancuran bahan.
  8. SPORADIS >< ...
    1. Terbagi - bagi
    2. Pemisahan
    3. Bersatu
    4. Jarang
    5. Sering
    Jawaban : E
    Sporadis = kadang - kadang. Lawan kata yang tepat untuk kata sporadis adalah kata sering.
  9. SUMBANG >< ....
    1. Derma
    2. Amal
    3. Lurus
    4. Benar
    5. Fiktif
    Jawaban : D
    Sumbang = Salah; Tidak selaras. Lawan kata yang tepat untuk kata sumbang adalah kata benar.
  10. AMATIR >< ....
    1. Palsu
    2. Canggih
    3. Ahli
    4. Partikelir
    5. Anasir
    Jawaban : C
    Lawan kata yang tepat untuk kata amatir adalah kata ahli.
