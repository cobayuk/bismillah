Contoh Soal dan Pembahasan TIU SKD bagian 17

Jun 11, 2020

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. ... : ... = KECEPATAN : KNOT
    1. Meter : Panjang
    2. Pipa : Asap
    3. Kapal : Pelabuhan
    4. Are : Dalam
    5. Jarak : Meter
    Jawaban : E
    Satuan pengukur kecepatan adalah knot. Satuan pengukur jarak adalah meter.
  2. BAWAH : ATAS = RINGAN : ...
    1. Jauh
    2. Gampang
    3. Berat
    4. Wajar
    5. Luas
    Jawaban : C
    Lawan kata dari bawah adalah atas. Lawan kata dari ringan adalah berat.
  3. KONVERSATORIUM : SENIMAN = ... : ...
    1. Akademi : Taruna
    2. Rumah sakit : Obat
    3. Ruang pengadilan : Saksi
    4. Seminar : Mahasiswa
    5. Rencana : Projeksi
    Jawaban : D
    Yang memanfaatkan konversatorium biasanya adalah para seniman. Yang memanfaatkan kegunaan seminar biasanya adalah para mahasiswa.
  4. UNDIAN : HADIAH = ... : ...
    1. Tuntutan : Santunan
    2. Persediaan : Panawaran
    3. Rusak : Ganti rugi
    4. Permintaan : Pemberian
    5. Tabungan : Pembelanjaan
    Jawaban : A
    Orang yang mengikuti undian bisa mendapatkan hadiah. Orang yang mengajukan tuntutan bisa mendapatkan santunan.
  5. JAMUR = ....
    1. Baksil
    2. Fungsi
    3. Bakteri
    4. Virus
    5. Enzim
    Jawaban : B
    Jamur = jenis tumbuhan yg tidak berdaun dan tidak berbuah, berkembang biak dengan spora, biasanya berbentuk payung, tumbuh di daerah berair atau lembap atau batang busuk; cendawan; kulat.
  6. TAMANAN = ....
    1. Menjalar
    2. Flora
    3. Menyerupai hampatan
    4. Rapat
    5. Ikan hias
    Jawaban : B
    Tamanan = ( 1 ) kebun yg ditanami dengan bunga - bunga dsb (tempat bersenang - senang); ( 2 ) tempat (yang menyenangkan dsb); ( 3 ) tempat duduk pengantin perempuan (yang dihiasi dengan bunga - bunga dsb).
  7. LEGITIMASI = ...
    1. Pengaturan
    2. Pengendalian
    3. Pengesahan
    4. Pengikat
    5. Perundingan
    Jawaban : C
    Legitimasi = (surat) keterangan yang mensahkan atau membenarkan atau menerangkan tahun pemegang surat itu benar - benar dia.
  8. POLIGAMI >< ...
    1. Tikus
    2. Wanita
    3. Cergam
    4. Monogami
    5. Sejenis
    Jawaban : D
    Poligami = Beristri banyak. Lawan kata yang tepat untuk kata poligami adalah kata monogami.
  9. LECO >< ....
    1. Kerdil
    2. Kecil
    3. Nakal
    4. Raksasa
    5. Tepat
    Jawaban : D
    Leco = Kerdil. Lawan kata yang tepat untuk kata leco adalah kata raksasa.
  10. MAYOR >< ...
    1. Junior
    2. Minim
    3. Mikro
    4. Minor
    5. Makro
    Jawaban : D
    Mayor = Besar. Lawan kata yang tepat untuk kata Mayor adalah kata minor.
