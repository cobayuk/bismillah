Contoh Soal dan Pembahasan TIU SKD bagian 19

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. TANAH LIAT : TEMBIKAR = ...
    1. Nasi : Beras
    2. Jagung : Sagu
    3. Pasir : Gelas
    4. Tembok : Semen
    5. Bulpen : Pensil
    Jawaban : C
    Tanah liat dapat diubah menjadi tembikar. Pasir dapat diubah menjadi gelas.
  2. BENSIN : MOBIL = ... : ...
    1. Listrik : Setrum
    2. Rumput : Ayam
    3. Cicak : Nyamuk
    4. Nasi : Orang
    5. Kambing : Rumput
    Jawaban : D
    Bensin merupakan bahan bakar utama sebuah mobil. Nasi merupakan makanan utama orang.
  3. PRODUKSI : USAHA = PRESTASI : ....
    1. Penjualan
    2. Penawaran
    3. Harga
    4. Hasil
    5. Prestasi
    Jawaban : D
    Produksi sama artinya dengan usaha. Prestasi sama artinya dengan hasil.
  4. JANTUNG : ... = OTAK : MEMIKIR
    1. Pompa
    2. Kecerdasan
    3. Sentra
    4. Kepala
    5. Syarat
    Jawaban : A
    Otak digunakan untuk berfikir. Jantung digunakan untuk memompa darah.
  5. PERAPIAN = ...
    1. Jamur
    2. Tanur
    3. Perkakas tukang
    4. Pakaian kuda
    5. Kotoran hewan
    Jawaban : B
    Perapian = tempat api; anglo; keran; tungku.
  6. PANGGUNG = ...
    1. Puji
    2. Dayung
    3. Anjung
    4. Buyung
    5. Angkat
    Jawaban : C
    Panggung = lantai; bangunan yang agak tinggi; pentas.
  7. OTONOMI = ...
    1. Daerah
    2. Swatantra
    3. Wilayah
    4. Distrik
    5. Sendiri
    Jawaban : B
    Otonomi = pemerintahan sendiri.
  8. NORMAL >< ...
    1. Tengah
    2. Pinggir
    3. Cacat
    4. Sesuai
    5. Standar
    Jawaban : C
    Normal = tidak ada kelainan. Lawan kata yang tepat untuk kata normal adalah kata cacat.
  9. CEMERLANG >< ...
    1. Berlian
    2. Dungu
    3. Terang
    4. Indah
    5. Cahaya
    Jawaban : B
    Cemerlang = Bersinar; cerdas. Lawan kata yang tepat untuk kata cemerlang adalah kata dungu.
  10. PLAGIAT >< ...
    1. Orisinil
    2. Turunan
    3. Implikasi
    4. Replikasi
    5. Penjiplak
    Jawaban : A
    Plagiat = Penjiplak. Lawan kata yang tepat untuk kata plagiat adalah kata orisinil.
