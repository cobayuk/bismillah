  1. Ringkih = ...
    1. Rapuh
    2. Sombong
    3. Kikir
    4. Daging
    5. Rempah
    Jawaban : A
    Ringkih = lemah; rapuh; tidak kokoh; tidak kuat.
  2. Ratifikasi = ...
    1. Cuma-cuma
    2. Pengesahan
    3. Uraian
    4. Penunjukan
    5. Pemberian
    Jawaban : B
    Ratifikasi = pengesahan suatu dokumen negara oleh parlemen, khususnya pengesahan undang-undang, perjanjian antarnegara, dan persetujuan hukum internasional.
  3. Rabat = ...
    1. Alat
    2. Teman
    3. Kikis
    4. Potongan harga
    5. Berita
    Jawaban : D
    Rabat = potongan harga.
  4. Pretensi = ...
    1. Nadi
    2. Jelas
    3. Nyata
    4. Alasan
    5. Pura-pura
    Jawaban : E
    Pretensi = keinginan yang kurang berdasar; perbuatan berpura-pura; alasan yang dibuat-buat.
  5. Preposisi = ...
    1. Kata depan
    2. Samping
    3. Sebelum
    4. Belakang
    5. Letak
    Jawaban : A
    Preposisi = kata yang biasa terdapat di depan nomina.
  6. Perlop = ...
    1. Absen
    2. Cuti
    3. Terlambat
    4. Masuk
    5. Pulang
    Jawaban : B
    Perlop = cuti (izin tidak bekerja dalam beberapa hari untuk beristirahat atau karena sakit dan sebagainya).
  7. Partikelir = ...
    1. Kecil
    2. Pemerintah
    3. Swasta
    4. Besar
    5. Ukuran
    Jawaban : C
    Partikelir = bukan untuk umum; bukan kepunyaan pemerintah; bukan (milik) dinas; swasta.
  8. Niscaya = ...
    1. Kabur
    2. Buram
    3. Percaya
    4. Pasti
    5. Hasil
    Jawaban : D
    Niscaya = tentu; pasti; tidak boleh tidak.
  9. Mutakhir = ...
    1. Ampuh
    2. Beda
    3. Sama
    4. Terkini
    5. Lama
    Jawaban : D
    Mutakhir = terakhir; terbaru; modern.
  10. Mobilitas = ...
    1. Kendaraan
    2. Alat
    3. Posisi
    4. Perangkat
    5. Gerak
    Jawaban : E
    Mobilitas = kesiapsiagaan untuk bergerak; gerakan berpindah-pindah.
