Contoh Soal dan Pembahasan TIU SKD bagian 20

Jun 11, 2020

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. AIR : BASAH = API : ...
    1. Panas
    2. Nyala
    3. Hangus
    4. Asap
    5. Merah
    Jawaban : C
    Jika terkena air akan menjadi basah. Jika terkena api akan menjadi hangus.
  2. PERGI : DATANG = JUAL : ...
    1. Dagang
    2. Konsumen
    3. Beli
    4. Produsen
    5. Bisnis
    Jawaban : C
    Lawan dari kata pergi adalah kata datang. Lawan dari kata jual adalah kata beli.
  3. TELUK : LAUT = ...
    1. Darat : Laut
    2. Utara : Selatan
    3. Semenanjung : Tanah
    4. Udara : Panas
    5. Langit : Bumi
    Jawaban : C
    Teluk adalah bagian laut yang menjorok ke daratan. Semenanjung adalah daratan atau tanah yang menjorok ke laut.
  4. GURU : MURID = ...
    1. Bapak : Ibu
    2. Mengaji : Uztad
    3. Orang tua : Anak
    4. Dokter : Pasien
    5. Arsitek : Taman
    Jawaban : D
    Guru memberi pelajaran kepada muridnya. Dokter memberi pengobatan kepada pasiennya.
  5. RINGKIH = ...
    1. Kuda
    2. Suara
    3. Rapuh
    4. Kuat
    5. Tinju
    Jawaban : C
    Ringkih = lrmah; rapuh; tidak kokoh; tidak kuat.
  6. HARTA BENDA = ...
    1. Mal
    2. Warisan
    3. Bencana
    4. Sedekah
    5. Zakat
    Jawaban : A
    Harta benda = barang kekayaan.
  7. RUJUKAN = ...
    1. Pacuan
    2. Acuan
    3. Pedoman
    4. Pemacu
    5. Peregangan
    Jawaban : B
    Rujukan = keterangan lanjutan mengenai suatu hal; acuan; referensi.
  8. PANCARONA > < ...
    1. Beragam
    2. Sekelompok
    3. Seragam
    4. Sepasang
    5. Lima macam
    Jawaban : C
    Pancarona = Beragam. Lawan kata yang tepat untuk kata pancarona adalah kata seragam.
  9. KAPABEL >< ...
    1. Ibu
    2. Pandai
    3. Bodoh
    4. Piawai
    5. Santai
    Jawaban : C
    Kapabel = Sanggup; mampu; pandai. Lawan kata yang tepat untuk kata kapabel adalah kata bodoh.
  10. ELUSIF >< ...
    1. Mudah dipahami
    2. Sukar ditembus
    3. Sukar ditiru
    4. Sukar dijalankan
    5. Mudah diikuti
    Jawaban : A
    Elusif = Susah dipahami. Lawan kata yang tepat untuk kata elusif adalah kata mudah dipahami.
