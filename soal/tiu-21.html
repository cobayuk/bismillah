Contoh Soal dan Pembahasan TIU SKD bagian 21

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. ... : ... = TULANG : LIGAMEN
    1. Senam : Pemanasan
    2. Lemak : Sel
    3. Kuku : Jari
    4. Otot : Tendon
    5. Lutut : Persendian
    Jawaban : D
    Tulang dihubungkan oleh ligamen. Otot dihubungkan oleh tendon.
  2. ... : ... = PICISAN : TAK BERNILAI
    1. Berlian : Kekayaan
    2. Palsu : Murah
    3. Kikir : Philantropi
    4. Mahal : Diskon
    5. Murah : Mahal
    Jawaban : C
    Kata picisan mempunyai arti sama dengan tak bernilai. Kata kikir mempunyai arti sama dengan philantropi.
  3. BELAJAR : PINTAR = ... : ...
    1. Menabung : Kaya
    2. Pelit : Boros
    3. Bekerja : Jujur
    4. Rumah : Bata
    5. Memancing : Memasak
    Jawaban : A
    Orang belajar agar menjadi pintar. Orang menabung agar menjadi kaya.
  4. JAKARTA : PERSIJA = ... : ...
    1. Persela : Lamongan
    2. Semarang : PSIS
    3. Surabaya : Persib
    4. Bandung : PSM
    5. Persipura : Makassar
    Jawaban : B
    Kota Jakarta memiliki klub sepakbola persija. Kota Semarang memiliki klub sepakbola PSIS.
  5. INTUISI = ...
    1. Ambang batas
    2. Kesetaraan
    3. Puncak
    4. Kesetimbangan
    5. Bisikan hati
    Jawaban : E
    Intuisi = daya atau kemampuan mengetahui atau memahami sesuatu tanpa dipikirkan atau dipelajari; bisikan hati; gerak hati.
  6. EKSPOSISI = ...
    1. Bahaya
    2. Keterangan
    3. Paparan
    4. Gambaran
    5. Ekspos
    Jawaban : C
    Eksposisi= uraian (paparan) yang bertujuan menjelaskan maksud dan tujuan.
  7. RAGIB = ...
    1. Pendeta
    2. Asyik
    3. Mahasiswa
    4. Petapa
    5. Suara
    Jawaban : B
    Ragib = suka sekali; asyik.
  8. IMITATIF >< ...
    1. Asli
    2. Pribumi
    3. Palsu
    4. Serupa
    5. Tiruan
    Jawaban : A
    Imitatif = Bersifat tiruan. Lawan kata yang tepat untuk kata imitatif adalah kata asli.
  9. MEGAT >< ...
    1. Tak kenal
    2. Palsu
    3. Sahih
    4. Serupa
    5. Damai
    Jawaban : A
    Megat = Terkenal. Lawan kata yang tepat untuk kata megat adalah kata tak kenal.
  10. PADAN >< ...
    1. Bukan dandanan
    2. Bukan bandingan
    3. Bukan majikan
    4. Bukan suruhan
    5. Sama
    Jawaban : B
    Padan = Sebanding. Lawan kata yang tepat untuk kata padan adalah kata bukan bandingan.
