Contoh Soal dan Pembahasan TIU SKD bagian 22

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. PAMAN : ... = KAKEK : CUCU
    1. Anak
    2. Kemanakan
    3. Buyut
    4. Cicit
    5. Ayah
    Jawaban : B
    Seorang dipanggil kakek oleh cucunya. Seorang dipanggil paman oleh kemanakannya.
  2. AYAH : JEJAKA = IBU : ...
    1. Pemimpin
    2. Pria
    3. Lajang
    4. Pemuda
    5. Gadis
    Jawaban : E
    Seorang jejaka akan menjadi seorang ayah. Seorang gadis akan menjadi seorang ibu.
  3. TELINGA : MENDENGAR = MATA : ...
    1. Dingin
    2. Panas
    3. Meraba
    4. Melihat
    5. Anting
    Jawaban : D
    Telinga digunakan untuk mendengar. Mata digunakan untuk melihat.
  4. BULATAN : ... = SEGI EMPAT : KUBUS
    1. Lingkaran
    2. Bola
    3. Balok
    4. Prisma
    5. Kerucut
    Jawaban : B
    Benda yang berbentuk segi empat adalah kubus. Benda yang berbentuk lingkaran adalah bola.
  5. PAILIT = ...
    1. Rugi
    2. Jual
    3. Hilang
    4. Rentan
    5. Bangkrut
    Jawaban : E
    Pailit = jatuh (tt perusahaan dsb); bangkrut.
  6. PENGIKISAN = ...
    1. Elaborasi
    2. Abolisi
    3. Abrasi
    4. Aborsi
    5. Erosi
    Jawaban : C
    Abrasi = pengikisan batuan oleh air, es, atau angin yang mengandung dan mengangkut hancuran bahan.
  7. SELEKSI = ...
    1. Jenis
    2. Macam
    3. Sortir
    4. Pilih
    5. Punah
    Jawaban : D
    Seleksi = pemilihan (untuk mendapatkan yang terbaik) .
  8. TAKZIM >< ...
    1. Hormat
    2. Yakin
    3. Patuh
    4. Lazim
    5. Lancang
    Jawaban : E
    Takzim = Hormat dan sopan. Lawan kata yang tepat untuk kata takzim adalah kata lancang.
  9. LONGGAR >< ...
    1. Sempit
    2. Kurang
    3. Kendur
    4. Kecil
    5. Sedang
    Jawaban : A
    Longgar = Lebar; tidak sempit. Lawan kata yang tepat untuk kata longgar adalah kata sempit.
  10. PERINTIS >< ...
    1. Pewaris
    2. Pelopor
    3. Penerus
    4. Pengikut
    5. Pencipta
    Jawaban : C
    Perintis = Pelopor; orang yang memulai mengerjakan sesuatu. Lawan kata yang tepat untuk kata perintis adalah kata penerus.
