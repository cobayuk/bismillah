Contoh Soal dan Pembahasan TIU SKD bagian 23

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. PISAU : TEMPA = PATUNG : ...
    1. Pahat
    2. Kayu
    3. Hiasan
    4. Seni
    5. Arca
    Jawaban : A
    Pisau dibuat dengan cara ditempa. Patung dibuat dengan cara dipahat.
  2. JUJUR : DUSTA = ... : ...
    1. Besar : Panjang
    2. Besar : Kecil
    3. Pendek : Kecil
    4. Sedang : Panjang
    5. Pendek : Besar
    Jawaban : B
    Lawan kata dari kata jujur adalah kata dusta. Lawan kata dari kata besar adalah kata kecil.
  3. PERAWAT : PASIEN = PEDAGANG : ...
    1. Langganan
    2. Barang
    3. Pembeli
    4. Usaha
    5. Konsumen
    Jawaban : C
    Tugas perawat adalah melayani pasien. Tugas pedagang adalah melayani pembeli.
  4. ROTAN : KURSI = KAYU : ...
    1. Atap
    2. Dinding
    3. Pintu
    4. Plastik
    5. Pohon
    Jawaban : C
    Rotan adalah bahan pembuat kursi. Kayu adalah bahan pembuat pintu.
  5. KAMPIUN = ...
    1. Tim
    2. Juara
    3. Group
    4. Kelompok
    5. Kalah
    Jawaban : B
    Kampiun = pemenang terakhir dl perlombaan atau pertandingan; juara.
  6. TENDER = ...
    1. Penawaran
    2. Kerja
    3. Harga
    4. Program
    5. Jumlah
    Jawaban : A
    Tender = tawaran untuk mengajukan harga, memborong pekerjaan, atau menyediakan barang.
  7. FLUKTUASI = ...
    1. Gerak
    2. Kredit
    3. Gejolak
    4. Musim
    5. Suku bunga
    Jawaban : C
    Fluktuasi = ( 1) gejala yg menunjukkan turun-naiknya harga; keadaan turun-naik harga dsb; perubahan (harga tsb) karena pengaruh permintaan dan penawaran; ( 2) ketidaktetapan; kegoncangan.
  8. GASAL >< ...
    1. Ganjil
    2. Genap
    3. Angka
    4. Gangsal
    5. Bilangan
    Jawaban : B
    Gasal = Ganjil. Lawan kata yang tepat untuk kata gasal adalah kata genap.
  9. INSOMNIA >< ...
    1. Susah makan
    2. Susah bangun
    3. Susah tidur
    4. Ngantuk
    5. Mudah tidur
    Jawaban : E
    Insomnia = Susah tidur. Lawan kata yang tepat untuk kata insomnia adalah kata mudah tidur.
  10. PARADOKSIAL >< ...
    1. Bertentangan
    2. Konflik
    3. Kontradiksi
    4. Sejalan
    5. Polaritas
    Jawaban : D
    Paradoksal = Tidak sejalan. Lawan kata yang tepat untuk kata paradkosal adalah kata sejalan.
