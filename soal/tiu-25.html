Contoh Soal dan Pembahasan TIU SKD bagian 25

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. JAM : ... = KALENDER : HARI
    1. Masehi
    2. Waktu
    3. Tahun
    4. Detik
    5. Jam
    Jawaban : D
    Kalender tersusun dari kumpulan hari. Jam tersusun dari kumpulan detik.
  2. ASING : ... = LOKASI : DOMESTIK
    1. Lokasi
    2. Manca
    3. Tradisional
    4. Domain
    5. Daerah
    Jawaban : B
    Lokal sama artinya dengan domestik. Asing sama artinya dengan manca.
  3. HEWAN : KAMBING = MAKANAN : ...
    1. Sate
    2. Air
    3. Waktu makan
    4. Lapar
    5. Kenyang
    Jawaban : A
    Salah satu jenis hewan adalah kambing. Salah satu jenis makanan adalah sate.
  4. PRESTASI : HASIL = PRODUKSI : ...
    1. Penawaran
    2. Penjualan
    3. Harga
    4. Usaha
    5. Prestasi
    Jawaban : D
    Prestasi adalah suatu hasil yang dicapai. Produksi adalah suatu usaha yang dilakukan.
  5. ISTAL = ...
    1. Perapian
    2. Jamur
    3. Istana
    4. Kandang kuda
    5. Pasang
    Jawaban : D
    Istal = kandang kuda.
  6. KELAKAR = ...
    1. Tertawa
    2. Goda
    3. Canda
    4. Senang
    5. Cacian
    Jawaban : C
    Kelakar = (1) perkataan yang bersifat lucu untuk membuat orang tertawa (gembira); lawak; canda.
  7. GALAT = ...
    1. Pendek
    2. Ciri khas
    3. Cacat
    4. Rata - rata
    5. Batas hitungan
    Jawaban : C
    Galat = kekeliruan, kesalahan; cacat.
  8. FIKSI >< ...
    1. Legenda
    2. Imajinasi
    3. Non fiksi
    4. Mitos
    5. Impian
    Jawaban : C
    Lawan kata yang tepat untuk kata fiksi adalah kata non fiksi.
  9. SKEPTIS >< ...
    1. Semangat
    2. Yakin
    3. Ragu - ragu
    4. Tidak yakin
    5. Perjuangan
    Jawaban : B
    Skeptis = Ragu - ragu. Lawan kata yang tepat untuk kata skeptis adalah kata yakin.
  10. MUSLIHAT >< ...
    1. Jujur
    2. Sombong
    3. Brutal
    4. Berani
    5. Parah
    Jawaban : A
    Muslihat = Tipuan. Lawan kata yang tepat untuk kata muslihat adalah kata jujur.
