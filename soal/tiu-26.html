Contoh Soal dan Pembahasan TIU SKD bagian 26

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. MAWAR : TANAMAN = ... : ...
    1. Pohon : Jati
    2. Anak : Ibu
    3. Kamar : Rumah
    4. Kucing : Tikus
    5. Singa : Binatang
    Jawaban : E
    Mawar termasuk dalam jenis tanaman. Singa termasuk dalam jenis binatang.
  2. AIR : UAP = ... : ...
    1. Mendidih : Mencair
    2. Membeku : Menguap
    3. Es : Air
    4. Kabut : Asap
    5. Uap : Beku
    Jawaban : C
    Air jika dipanaskan akan menjadi uap. Es jika dipanaskan akan menjadi air.
  3. KAYU : ... = BERAS : NASI GORENG
    1. Pasak
    2. Kursi tamu
    3. Gelondongan
    4. Papan
    5. Tripleks
    Jawaban : B
    Beras jika dimasak bisa menjadi nasi goreng. Kayu jika diolah bisa menjadi kursi tamu.
  4. MIKROSKOP : BAKTERIOLOG = SARUNG TANGAN : ...
    1. Apoteker
    2. Petinju
    3. Dokter
    4. Analis
    5. Peneliti
    Jawaban : B
    Mikroskop adalah alat yang digunakan bakteriologi. Sarung tangan adalah alat yang digunakan oleh petinju.
  5. GETIR = ...
    1. Manis
    2. Sakit
    3. Pedas
    4. Nyeri
    5. Pahit
    Jawaban : E
    Getir = rasa pahit agak pedas (seperti rasa kulit jeruk).
  6. BANDELA = ...
    1. Peti kemas
    2. Bendera
    3. Lambang
    4. Simbol
    5. Umbul - umbul
    Jawaban : A
    Bandela = bungkusan (peti) besar barang - barang; peti kemas.
  7. MUKADIMAH = ...
    1. Atas
    2. Pengantar
    3. Penutup
    4. Isi
    5. Pengganti
    Jawaban : B
    Mukadimah = (kata) pendahuluan; (kata) pengantar.
  8. SINKRON >< ...
    1. Selaras
    2. Serasi
    3. Berbeda
    4. Harmonis
    5. Sesuai
    Jawaban : C
    Sinkron = sesuai; sama. Lawan kata yang tepat untuk kata sinkron adalah berbeda.
  9. NEKAT >< ...
    1. Niat
    2. Motif
    3. Maksud
    4. Berani
    5. Takut
    Jawaban : E
    Nekat = Terlalu berani; tidak berpikir lagi. Lawan kata yang tepat untuk kata nekat adalah kata takut.
  10. ELEKTRIK >< ...
    1. Tunjuk
    2. Campur
    3. Kolot
    4. Pilih
    5. Tak pilih - pilih
    Jawaban : E
    Elektrik = Pilih - pilih. Lawan kata yang tepat untuk kata elektrik adalah tak pilih - pilih.
