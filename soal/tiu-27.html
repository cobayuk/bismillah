Contoh Soal dan Pembahasan TIU SKD bagian 27

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. BURUNG : SARANG = ... : ...
    1. Orang : Rumah
    2. Murid : Sekolah
    3. Dompet : Uang
    4. Kandang : Kambing
    5. Rapat : Gedung
    Jawaban : A
    Tempat tinggal burung adalah sarang. Tempat tinggal orang adalah rumah.
  2. SELASA : KAMIS = ... : ...
    1. Januari : Maret
    2. Menit : Jam
    3. Jam : Hari
    4. Mei : Agustus
    5. Gelap : Terang
    Jawaban : A
    Dua hari setelah hari Selasa adalah hari Kamis. Dua bulan setelah bulan Januari adalah bulan Maret.
  3. DINIHARI : SENJA = ... : ...
    1. Kakak : Anak
    2. Barat : Selatan
    3. Siang : Malam
    4. Sepatu : Tas
    5. Menit : Detik
    Jawaban : C
    Lawan dari dinihari adalah senja. Lawan dari siang malam.
  4. BURUNG : EKOR = ... : ...
    1. Gajah : Belalai
    2. Kapal : Buritan
    3. Kucing : Bulu
    4. Harimau : Taring
    5. Rusa : Tanduk
    Jawaban : B
    Yang terletak di bagian belakang burung adalah ekor. Yang terletak di bagian belakang kapal adalah buritan.
  5. DELUSI = ...
    1. Kekecewaan
    2. Khawatir
    3. Ilusi
    4. Fiksi
    5. Nyata
    Jawaban : C
    Delusi = pikiran atau pandangan yang tidak berdasar (tidak rasional); pendapat yang tidak berdasakan kenyataan; khayal. Khayal = ilusi.
  6. MEMANGKAS = ...
    1. Memotong
    2. Mati
    3. Menderita
    4. Pohon
    5. Ganjil
    Jawaban : A
    Memangkas = (1) memotong ujung (tumbuh - tumbuhan, dsb); (2) menggunting (rambut).
  7. TANGGALKAN = ...
    1. Catatan
    2. Lepaskan
    3. Mingguan
    4. Almanak
    5. Buluh
    Jawaban : B
    Tanggal = terlepas; luruh. Tanggalkan = lepaskan.
  8. IMIGRASI >< ...
    1. Emigrasi
    2. Migrasi
    3. Transportasi
    4. Transparansi
    5. Koalisi
    Jawaban : A
    Imigrasi = perpindahan penduduk negara lain (masuk) ke negara tertentu untuk menetap. Lawan katanya adalah emigrasi ( pindah/ keluar dari tanah air sendiri dan menetap di negara lain ).
  9. INDUKSI >< ...
    1. Reduksi
    2. Konduksi
    3. Deduksi
    4. Residu
    5. Redusi
    Jawaban : C
    Induksi = penarikan kesimpulan berdasarkan keadaan yang khusus untuk dperlakukan secara umum. Lawan katanya deduksi = penyimpulan dari keadaan yang umum ke khusus.
  10. NOMADIK >< ...
    1. Berpindah - pindah
    2. Perlahan - lahan
    3. Terus - menerus
    4. Menetap
    5. Datang
    Jawaban : D
    Nomadik = berpindah - pindah, lawannya adalah menetap.
