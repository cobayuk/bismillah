Contoh Soal dan Pembahasan TIU SKD bagian 28

Jun 11, 2020

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. PIZZA : GANDUM = ... : ...
    1. Rumah : Tukang
    2. Gambar : Pelukis
    3. Genteng : Tanah Liat
    4. Skripsi : Buku
    5. Patung : Pemahat
    Jawaban : C
    Bahan membuat pizza adalah gandum. Bahan membuat genteng adalah tanah liat.
  2. MODERN : TRADISIONAL = ... : ...
    1. Roket : Rudal Scud
    2. Mobil : Pedati
    3. Ferrari : Fiat
    4. TV : VCD Player
    5. Pesawat : Sepeda Motor
    Jawaban : B
    Mobil alat transportasi modern, pedati alat transportasi tradisional.
  3. INDONESIA : BELANDA = ... : ...
    1. Malaysia : Inggris
    2. India : Spanyol
    3. Australia : Selandia Baru
    4. Singapura : China
    5. Brunei Darussalam : Timor Leste
    Jawaban : A
    Indonesia pernah dijajah Belanda. Malaysia pernah dijajah Inggris.
  4. KARNIVORA : SINGA = ... : ...
    1. Reptilia : Buaya
    2. Manusia : Omnivora
    3. Herbivora : Sapi
    4. Omnivora : Harimau
    5. Herbivora : Omnivora
    Jawaban : C
    Contoh hewan karnivora adalah singa. Contoh hewan herbivora adalah sapi.
  5. GANCU = ...
    1. Serokan
    2. Karung
    3. Timbangan
    4. Pengait
    5. Jangkar
    Jawaban : D
    Gancu = sejenis alat terbuat dari batang besi yang ujungnya diberi pengait. Alat ini biasanya digunakan untuk menarik karung beras.
  6. KUNYUK = ...
    1. Sial
    2. Macan
    3. Gorila
    4. Kera kecil
    5. Kancil
    Jawaban : D
    Kunyuk = kera kecil, monyet; kata sindiran untuk menyatakan orang yang tidak tahu aturan (adat) atau orang bodoh.
  7. KONVENSI = ...
    1. Musyawarah
    2. Kesepakatan
    3. Ketentuan
    4. Perundingan
    5. Syarat
    Jawaban : B
    Konvensi = kesepakatan umum terutama mengenai bentuk - bentuk tata krama; adat istiadat atau kebiasaan yang berdasarkan kesepakatan umum.
  8. PERLOP >< ...
    1. Masuk
    2. Kerja
    3. Datang
    4. Absen
    5. Pulang
    Jawaban : B
    Perlop = cuti (berhenti kerja beberapa lama untuk istirahat/ rehat, karena sakit, dsb.). Lawannya adalah kerja.
  9. TESIS >< ...
    1. Sintetis
    2. Hipotesis
    3. Epitesis
    4. Antitesis
    5. Paratesis
    Jawaban : D
    Tesis adalah pernyataan atau teori untuk dikemukakan dan didukung oleh argumen - argumen. Lawannya adalah antitesis = pernyataan yang bertentangan dengan kebenaran (biasanya tidak di dukung teori atau argumen - argumen).
  10. PEMUPUKAN >< ...
    1. Reboisasi
    2. Defertilisasi
    3. Renovasi
    4. Rehabilitasi
    5. Fertilisasi
    Jawaban : B
    Pemupukan = tindakan menyuburkan tanah dengan pupuk (rabuk). Lawannya adalah defertilisasi = keadaan tidak subur (kurang pupuk).
