Contoh Soal dan Pembahasan TIU SKD bagian 29

Jun 11, 2020

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. ACEH : SERAMBI MEKAH = ... : ...
    1. Michael Jackson : King Of Pop
    2. Michael Schumacer : Pembalap
    3. Michael Gorbachev : Pemimpin Soviet
    4. Madona : Penari
    5. Agnes Monica : Penyanyi
    Jawaban : A
    Julukan untuk Aceh adalah Serambi Mekah. Julukan untuk Michael Jackson adalah King of pop.
  2. DOKTOR : DISERTASI = ... : ...
    1. Buruh : Upah
    2. Menteri : Kepmen
    3. Kyai : Jamaah
    4. Sarjana : Skripsi
    5. Penyanyi : Lagu
    Jawaban : D
    Syarat menjadi doktor adalah membuat disertasi. Syarat menjadi sarjana adalah membuat skripsi.
  3. GEMPA : RITCHER = ... : ...
    1. Suhu : Fahrenheit
    2. Ombak : Knot
    3. Jarak : Dinamo
    4. Banjir : Air
    5. Panas : Derajat
    Jawaban : A
    Gempa diukur dengan skala Ritcher. Suhu diukur dengan skala Fahrenheit.
  4. PARI : IKAN = ... : ...
    1. Gandum : Teri
    2. Bayam : Sayur
    3. Mangga : Manis
    4. Burung : Ayam
    5. Jambu : Biji
    Jawaban : B
    Pari merupakan jenis ikan. Bayam merupakan jenis sayur.
  5. ENTITAS = ...
    1. Jumlah
    2. Wujud
    3. Tak terbatas
    4. Sumbangsih
    5. Batasan
    Jawaban : B
    Entitas = satuan yang berwujud, wujud.
  6. IMPRESI = ...
    1. Pesan
    2. Kiasan
    3. Kesan
    4. Wujud
    5. Peran
    Jawaban : C
    Impresi = pendapat, kesan.
  7. BABUT = ...
    1. Tikar
    2. Sprei
    3. Karpet
    4. Permadani
    5. Selimut
    Jawaban : D
    Babut = berasal dari bahasa Jawa yang artinya permadani.
  8. KAPABEL >< ...
    1. Piawai
    2. Rajin
    3. Pandai
    4. Bodoh
    5. Rapi
    Jawaban : D
    Kapabel = cakap; pandai; sanggup. Lawannya adalah bodoh.
  9. SIAU >< ...
    1. Memuai
    2. Membeku
    3. Mendidih
    4. Memanas
    5. Menguap
    Jawaban : C
    Siau = sudah tak panas lagi(air panas, dsb.); dingin kembali (panas tubuh). Lawannya adalah mendidih.
  10. LECO >< ...
    1. Raksasa
    2. Mungil
    3. Kerdil
    4. Cebol
    5. Kurcaci
    Jawaban : A
    Leco = kerdil; cebol; kate; kecil. Lawannya adalah raksasa.
