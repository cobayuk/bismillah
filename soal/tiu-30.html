Contoh Soal dan Pembahasan TIU SKD bagian 30

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. KOMPOR : API = ... : ...
    1. Kipas : Angin
    2. Lemari : Es
    3. Pohon : Buah
    4. Jalan : Macet
    5. Sendal : Kaki
    Jawaban : A
    Kompor memerlukan api untuk bisa berfungsi. Kipas membutuhkan angin agar berfungsi.
  2. SEPATU : JALAN = ... : ...
    1. Pensil : Makan
    2. Garpu : Makan
    3. Sisir : Rambut
    4. Buku : Baca
    5. Cuci : Baju
    Jawaban : B
    Sepatu digunakan untuk berjalan, dan garpu digunakan untuk makan.
  3. MOBIL : BENSIN = ... : ...
    1. Sapi : Susu
    2. Pesawat Terbang : Propeler
    3. Manusia : Makan
    4. Penyakit : Virus
    5. Ngantuk : Tidur
    Jawaban : C
    Mobil membutuhkan bensin. Manusia membutuhkan makanan.
  4. PIANO : ORGAN = ... : ...
    1. Nada : Not
    2. Senar : Pipa
    3. Kunci : Pedal
    4. Drum : Gitar
    5. Panci : Wajan
    Jawaban : A
    Piano mirip dengan organ. Nada mirip dengan not.
  5. AMBAL = ...
    1. Rial
    2. Parade
    3. Barisan
    4. Pawai
    5. Riuh
    Jawaban : D
    Ambal = sebuah istilah dalam sastra lama yang artinya berarak - arak; biasanya digunakan untuk mengungkapkan suatu iring - iringan barisan atau berpawai.
  6. DEMAGOGI = ...
    1. Menghasut
    2. Menghina
    3. Membela
    4. Mempelajari
    5. Membujuk
    Jawaban : A
    Demagogi = istilah dalam politik untuk menyatakan atau memperoleh kekuasaan dengan jalan menghasut dan membangkitkan emosi rakyat dalam pidato - pidatonya.
  7. LEJAR = ...
    1. Lemas
    2. Lemah
    3. Penat
    4. Lesu
    5. Lalai
    Jawaban : C
    Lejar = suatu keadaan dimana seseorang merasa sangat penat (letih) sehabis melakukan sesuatu.
  8. TELAGA >< ...
    1. Lautan
    2. Sumur
    3. Danau
    4. Mata Air
    5. Arus
    Jawaban : A
    Telaga = genangan air yang cukup luas dan dalam di daratan, lawannya adalah genangan air yang cukup luas yang melingkupi daratan (laut/ lautan).
  9. ROBOH >< ...
    1. Runtuh
    2. Miring
    3. Hancur
    4. Tegak
    5. Rusak
    Jawaban : D
    Roboh = (1) runtuh (barang yang besar seperti rumah, tembok, dsb.); (2) jatuh, rebah (pohon - pohonan, tanaman, dsb.) => jadi, lawannya ROBOH adalah tegak.
  10. DAWAI >< ...
    1. Tali
    2. Tambang
    3. Kawat
    4. Pita
    5. Sabut
    Jawaban : B
    Dawai = kawat yang halus. => jadi, lawannya adalah tambang.
