Contoh Soal dan Pembahasan TIU SKD bagian 31

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. FILM : SKENARIO = ... : ...
    1. Drama : Panggung
    2. Sandiwara : Plot
    3. Teater : Acara
    4. Resital : Tinjauan
    5. Pameran : Pengunjung
    Jawaban : B
    Skenario bagian dari film, plot bagian dari sandiwara.
  2. KEPAK : SAYAP = ... : ...
    1. Hirup : oksigen
    2. Sandar : kepala
    3. Hentak : kaki
    4. Tarik : tali
    5. Makan : roti
    Jawaban : C
    Sayap dikepakkan agar bisa terbang, kaki dihentakkan agar bisa jalan.
  3. DAUN : DURI = TERATAI : ...
    1. Ranting
    2. Kapur
    3. Kamboja
    4. Kaktus
    5. Air
    Jawaban : D
    Teratai mempunyai daun, sepola dengan kaktus mempunyai duri.
  4. IKAN MAS : IKAN PAUS = INSANG : ...
    1. Rusa
    2. Paru - paru
    3. Anjing
    4. Air Tawar
    5. Ikan
    Jawaban : B
    Ikan Mas : Ikan Paus => jenis - jenis ikan sepola dengan jenis - jenis alat pernapasan, => Insang : Paru - paru.
  5. ELITIS = ...
    1. Terpandang
    2. Terbatas
    3. Terpercaya
    4. Terbalik
    5. Terpadu
    Jawaban : A
    Elitis = (1) terpilih, terpandang (tentang kelompok dari masyarakat); (2) berhubungan dengan kelompok elite dari masyarakat: komunikasi merupakan syarat esensial bagi terciptanya prasyarat tumbuhnya kegiatan bersama yang terbuka, yang tidak bersifat elitis.
  6. AMBIGUITAS = ...
    1. Pengartian
    2. Penyimpangan
    3. Ketidakpastian
    4. Kebingungan
    5. Penyelewengan
    Jawaban : D
    Ambiguitas = (1) sifat atau hal yang bermakna dua; kemungkinan yang mempunyai dua pengertian; (2) ketidaktentuan, ketidakjelasan; (3) kemungkinan adanya makna atau penafsiran yang lebih dari satu atas suatu karya sastra; (4) Ling. Kemungkinan adanya makna lebih dari satu pada sebuah kata, gabungan kata, atau kalimat; ketaksaan.
  7. RELUNG = ...
    1. Ruang
    2. Labirin
    3. Dalam
    4. Panjang
    5. Lebar
    Jawaban : A
    Relung = lekuk atau jeluk (di tanah, di lereng gunung), ukuran luas tanah (ada yang kira - kira 5.300 m3 dan ada yang kira - kira 2.800 m3).
  8. RAPUH >< ...
    1. Tegang
    2. Teguh
    3. Lurus
    4. Sehat
    5. Remuk
    Jawaban : B
    Rapuh = (1) sudah rusak (patah, pecah sobek, putus); (2) lemah, sakit - sakitan (tentang tubuh); (3) tidak teguh (tidak tetap pendirian dsb.) => jadi, lawan dari rapuh adalah teguh.
  9. TENTATIF >< ...
    1. Tepat
    2. Jelas
    3. Pasti
    4. Langsung
    5. Cepat
    Jawaban : C
    Tentatif = (1) belum pasti; masih dapat berubah; (2) sementara waktu. => jadi, lawan dari TENTATIF adalah pasti.
  10. QUASI >< ...
    1. Nyata
    2. Langsung
    3. Lemah
    4. Turun
    5. Cepat
    Jawaban : A
    Quasi = hampir seperti; seolah - olah. => jadi, lawannya QUASI adalah nyata.
