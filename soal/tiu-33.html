Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. HUTAN : POHON = ... : ...
    1. Armada : Kapal
    2. Pohon : Papan
    3. Mawar : Duri
    4. Kamera : Film
    5. Sabun : Cuci
    Jawaban : A
    Hutan terdapat pohon. Armada terdapat kapal.
  2. LUCU : TERTAWA = ... : ...
    1. Kaget : Terkejut
    2. Cantik : Terpesona
    3. Sukses : Tercapai
    4. Hati : Tergerak
    5. Sedih : Terharu
    Jawaban : B
    Lucu membuat kita tertawa. Cantik membuat kita terpesona.
  3. PENNISILIN : FLEMMING = BAROMETER : ...
    1. Lavoiser
    2. Marconi
    3. Archimedes
    4. Toricelli
    5. Machiaveli
    Jawaban : D
    Pennisilin ditemukan oleh Flemming. Barometer ditemukan oleh Toricelli.
  4. KONSTITUSI : PREAMBULE = ... : ...
    1. Opera : Intermeso
    2. Legislatif : Pendahuluan
    3. Drama : Epilog
    4. Buku : Kata Pengantar
    5. Rumah : Halaman
    Jawaban : D
    Konstitusi diawali dengan preambul. Buku diawali dengan kata pengantar.
  5. PEDAR = ...
    1. Pisah
    2. Encer
    3. Getir
    4. Tajam
    5. Tandus
    Jawaban : C
    Pedar = (1) getir; (2) tengik (tentang rasa dan bau).
  6. AGUN = ...
    1. Gadai
    2. Hutang
    3. Pinjam
    4. Simpan
    5. Jual
    Jawaban : A
    Agun = cagar(an); tanggungan; jaminan.
  7. INGSUT = ...
    1. Bergeser
    2. Keluar
    3. Jauh
    4. Perlahan
    5. Langsung
    Jawaban : A
    Ingsut = bergeser sedikit.
  8. INSINUASI >< ...
    1. Terang - terangan
    2. Caci maki
    3. Rayuan
    4. Pujian
    5. Sapaan
    Jawaban : A
    Insinuasi = tuduhan terselubung, sindiran > < terang - terangan.
  9. GENERIK >< ...
    1. Mahal
    2. Khusus
    3. Ampun
    4. Individu
    5. Murah
    Jawaban : B
    Generik = umum, lazim >< khusus.
  10. PUJIAN >< ...
    1. Hadiah
    2. Sanjungan
    3. Cemoohan
    4. Cacian
    5. Apresiasi
    Jawaban : D
    Pujian = pernyataan penghargaan pada sesuatu yang membuatnya keheranan atau takjub; lawannya cacian atau sumpah serapah.
