Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. Hedonisme = ...
    1. Hemat
    2. Bahagia
    3. Semangat
    4. Gratis
    5. Hura-hura
    Jawaban : E
    Hedonisme = pandangan yang menganggap kesenangan dan kenikmatan materi sebagai tujuan utama dalam hidup.
  2. Getir = ...
    1. Pahit
    2. Alam
    3. Sakit
    4. Lidah
    5. Liar
    Jawaban : A
    Getir = rasa pahit agak pedas.
  3. Evokasi = ...
    1. Dendam
    2. Penggugah rasa
    3. Pengaruh
    4. Desak
    5. Buah
    Jawaban : B
    Evokasi = (daya) penggugah rasa.
  4. Afeksi = ...
    1. Tular
    2. Pengaruh
    3. Cita Rasa
    4. Bahagia
    5. Kasih sayang
    Jawaban : E
    Afeksi = rasa kasih sayang; erasaan dan emosi yang lunak.
  5. Abolisi >< ...
    1. Pemberatan
    2. Penangkapan
    3. Pembusukan
    4. Perubahan
    5. Pengawasan
    Jawaban : A
    Abolisi = peniadaan peristiwa pidana; penghapusan (perbudakan di Amerika);
    Jadi, lawan kata 'abolisi' adalah pemberatan.
  6. Anomali >< ...
    1. Aturan
    2. Normal
    3. Patuh
    4. Takut
    5. Salah
    Jawaban : B
    Anomali = ketidaknormalan; penyimpangan dari normal; kelainan; penyimpangan atau kelainan.
    Jadi lawan kata 'anomali' adalah 'normal'.
  7. Apatis >< ...
    1. Keluar
    2. Tanya
    3. Aktif
    4. Sombong
    5. Kikir
    Jawaban : C
    Apatis = acuh tidak acuh; tidak peduli; masa bodoh. Jadi lawan kata 'apatis' adalah 'aktif'.
  8. Delusi >< ...
    1. Khayalan
    2. Nurani
    3. Jiwa
    4. Nyata
    5. Kabur
    Jawaban : D
    Delusi = pikiran atau pandangan yang tidak berdasar (tidak rasional), biasanya berwujud sifat kemegahan diri atau perasaan dikejar-kejar; pendapat yang tidak berdasarkan kenyataan; khayal. Jadi lawan
  9. Elusif >< ...
    1. Khusus
    2. Terbatas
    3. Terbaru
    4. Mudah dimengerti
    5. Terbaru
    Jawaban : D
    Elusif = sukar dipahami atau diartikan; sukar diidentifikasi. Jadi lawan kata 'elusif' adalah 'mudah dimengerti'.
  10. Esoteris >< ...
    1. Dalam
    2. Aman
    3. Terbatas
    4. Cokelat
    5. Terbuka
    Jawaban : E
    Esoteris = bersifat khusus (rahasia, terbatas). Jadi lawan kata 'esoteris' adalah 'terbuka'.
