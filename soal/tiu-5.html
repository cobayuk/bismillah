Contoh Soal dan Pembahasan TIU SKD bagian 5

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. Hayati >< ...
    1. Mati
    2. Hidup
    3. Tumbuhan
    4. Sakit
    5. Demam
    Jawaban : A
    Hayati = Hidup. Jadi lawan kata 'hayati' adalah 'mati'.
  2. Hiperbola >< ...
    1. Olahraga
    2. Apa adanya
    3. Penjaga
    4. Lebih
    5. Kurang
    Jawaban : B
    Hiperbola = melebih-lebihkan. Jadi lawan kata 'hiperbola' adalah 'apa adanya'.
  3. Kapabel >< ...
    1. Panjang
    2. Bodoh
    3. Cakap
    4. Sanggup
    5. Pintar
    Jawaban : B
    Kapabel = mampu; cakap; pandai; sanggup. Jadi lawan kata 'kapabel' adalah 'bodoh'.
  4. Kekang >< ...
    1. Hewan
    2. Bebas
    3. Batas
    4. Kebas
    5. Sampai
    Jawaban : B
    Kekang = segala sesuatu yang dipakai untuk mencegah perbuatan yang diangggap kurang baik; larangan. Jadi lawan kata 'kekang' adalah 'bebas'.
  5. Kolektif >< ...
    1. Pasif
    2. Aktif
    3. Kumpul
    4. Individual
    5. Rangkaian
    Jawaban : D
    Kolektif = secara bersama; secara gabungan. Jadi lawan kata 'kolektif' adalah 'individual'.
  6. Konklusi >< ...
    1. Deskripsi
    2. Narasi
    3. Preposisi
    4. Uraian
    5. Akhir
    Jawaban : D
    Konklusi = simpulan (pendapat). Jadi lawan kata 'konklusi' adalah 'simpulan'.
  7. Kontan >< ...
    1. Awal
    2. Bayar
    3. Refleks
    4. Reaksi
    5. Hutang
    Jawaban : E
    Kontan = membayar tunai (tidak mengangsur atau bertangguh); membayar sekaligus (tidak meminjam atau mengangsur). Jadi lawan kata 'kontan' adalah 'hutang'.
  8. Kontra >< ...
    1. Perjanjian
    2. Gagal
    3. Setuju
    4. Debat
    5. Adil
    Jawaban : C
    Kontra = dalam keadaan tidak setuju; dalam keadaan menentang; menentang (pendapat dan sebagainya). Jadi lawan kata 'kontra' adalah 'setuju'.
  9. Konveks >< ...
    1. Nyata
    2. Cekung/konkaf
    3. Bias
    4. Lurus
    5. Gelap
    Jawaban : B
    Konveks = cembung. Jadi lawan kata 'konveks' adalah 'cekung'.
  10. Krusial >< ...
    1. Untung
    2. Genting
    3. Gawat
    4. Sepele
    5. Khawatir
    Jawaban : D
    Krusial = gawat; genting; menentukan. Jadi lawan kata 'krusial' adalah 'sepele'.
