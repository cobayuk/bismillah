Contoh Soal dan Pembahasan TIU SKD bagian 6

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. Tambun >< ...
    1. Tempat
    2. Diet
    3. Lemak
    4. Daging
    5. Kurus
    Jawaban : E
    Tambun = gemuk (tentang manusia dan binatang); berisi (tentang tubuh). Jadi lawan kata 'tambun' adalah 'kurus'.
  2. Lancung >< ...
    1. Asli
    2. Palsu
    3. Tulen
    4. Garansi
    5. Bekas
    Jawaban : A
    Lancung = tidak tulen; palsu; tiruan: Jadi lawan kata 'lancung' adalah 'asli'.
  3. Makar >< ...
    1. Busuk
    2. Jujur
    3. Emas
    4. Tanaman
    5. Kaki
    Jawaban : B
    Makar = akal busuk; tipu muslihat. Jadi lawan kata 'makar' adalah 'jujur'.
  4. Merana >< ...
    1. Kabar
    2. Sehat
    3. Derita
    4. Sifat
    5. Warna
    Jawaban : B
    Merana = lama menderita sakit; selalu sakit-sakit; makin lama makin kurus. Jadi lawan kata 'merana' adalah 'sehat'.
  5. Monoton >< ...
    1. Lihat
    2. Bioskop
    3. Statis
    4. Diam
    5. Berubah-ubah
    Jawaban : E
    Monoton = selalu sama dengan yang dulu; itu-itu saja, tidak ada ragamnya. Jadi lawan kata 'monoton' adalah 'berubah-ubah'.
  6. Ofensif >< ...
    1. Dalam
    2. Sering
    3. Bertahan
    4. Ulang
    5. Maju
    Jawaban : C
    Ofensif = bersifat maju dan menyerang dalam suatu kegiatan. Jadi lawan kata 'ofensif' adalah 'bertahan'.
  7. Pakar >< ...
    1. Bagian
    2. Tumbuhan
    3. Kenal
    4. Bawah
    5. Awam
    Jawaban : E
    Pakar = (orang) ahli; spesialis. Jadi lawan kata 'pakar' adalah 'awam'.
  8. Pasca >< ...
    1. Kemarin
    2. Sejarah
    3. Besok
    4. Pra
    5. Dulu
    Jawaban : D
    Pasca = sesudah. Jadi lawan kata 'pasca' adalah 'pra/sebelum'.
  9. Polemik >< ...
    1. Debat
    2. Rukun
    3. Kasar
    4. Berkelahi
    5. Masalah
    Jawaban : B
    Polemik = perdebatan mengenai suatu masalah yang dikemukakan secara terbuka dalam media massa. Jadi lawan kata 'polemik' adalah 'rukun'.
  10. Preambul >< ...
    1. Penjelasan
    2. Pembuka
    3. Penutup
    4. Tengah
    5. Awal
    Jawaban : C
    Preambul = mukadimah; pendahuluan. Jadi lawan kata 'preambul' adalah 'penutup'.
