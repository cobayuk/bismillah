Contoh Soal dan Pembahasan TIU SKD bagian 7

Jun 11, 2020

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. Prefiks >< ...
    1. Awalan
    2. Tambahan
    3. Sisipan
    4. Akhiran
    5. Hapus
    Jawaban : D
    Prefiks = imbuhan yang ditambahkan pada bagian awal sebuah kata dasar atau bentuk dasar; awalan. Jadi lawan kata 'prefiks' adalah 'akhiran'.
  2. Prominen >< ...
    1. Awal
    2. Setelah
    3. Populer
    4. Bayi
    5. Biasa
    Jawaban : E
    Prominen = terkemuka (tentang orang). Jadi lawan kata 'prominen' adalah 'biasa'.
  3. Remisi >< ...
    1. Pengurangan
    2. Penambahan
    3. Penghapusan
    4. Perencanaan
    5. Pelaksanaan
    Jawaban : B
    Remisi = pengurangan hukuman yang diberikan kepada orang yang terhukum. Jadi lawan kata 'remisi' adalah 'penambahan'.
  4. Siau >< ...
    1. Cahaya
    2. Panas
    3. Lampu
    4. Udara
    5. Mendidih
    Jawaban : E
    Siau = sudah tidak panas lagi (tentang air panas dan sebagainya); dingin kembali (tentang panas tubuh). Jadi lawan kata 'siau' adalah 'mendidih'.
  5. LAMPU : API = _____ : _____
    1. Palung : Gua
    2. Pohon : Buah
    3. Kipas : Angin
    4. Jalan : Macet
    5. Lemari : Es
    Jawaban : C
    Kompor memerlukan api untuk dapat berfungsi, dan kipas membutuhkan angin supaya dapat berfungsi.
  6. TELUR : BUTIR = _____ : _____
    1. Buku : Lembar
    2. Air : Meter
    3. Pakaian : Kodi
    4. Kertas : Kilogram
    5. Bawang : Siung
    Jawaban : E
    Telur dengan butir, sedangkan bawang dihitung dengan siung.
  7. BERPIKIR : ARIF = _____ : _____
    1. Kamera : Potret
    2. Bodoh : Rajin
    3. Belajar : Pandai
    4. Litografi : Batu
    5. Cetak : Kertas
    Jawaban : C
    Jika banyak berpikir kita menjadi arif. Jika banyak belajar kita menjadi pandai.
  8. KERTAS : HELAI = _____ : _____
    1. Jam : Detik
    2. Umur : Tahun
    3. Air : Minum
    4. Ilmu : Nilai
    5. Kecepatan : Jam
    Jawaban : B
    Kertas dihitung per helai, sedangkan umur dihitung per tahun.
  9. MANUSIA : MAKANAN = _____ : _____
    1. Sapi : Daging
    2. Sepeda : Solar
    3. Pesawat terbang : Penumpang
    4. Penyakit : Virus
    5. Mobil : Bensin
    Jawaban : E
    Manusia membutuhkan makanan, sedangkan mobil membutuhkan bensin.
  10. MENANAM : MENUAI = _____ : _____
    1. Makan : Kenyang
    2. Minum : Penuh
    3. Mencontek : Menghukum
    4. Menyakiti : Menangis
    5. Karya tulis : Jiplak
    Jawaban : A
    Setelah menanam akan menuai, setelah makan akan kenyang.
