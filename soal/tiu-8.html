Contoh Soal dan Pembahasan TIU SKD bagian 8

Jun 11, 2020

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. KARDIOLOGI : JANTUNG = _____ : _____
    1. Bakteriologi : Bakteri
    2. Obat-obatan : Farmakologi
    3. Ternologi : Termin
    4. Akutansi : Dagang
    5. Phatologi : Peta
    Jawaban : A
    Kardiologi adalah ilmu yang mempelajari tentang jantung, dan Bakteriologi adalah ilmu yang mempelajari tentang bakteri.
  2. SUHU : CELCIUS = _____ : _____
    1. Termo : Suhu
    2. Api : Kebakaran
    3. Banjir : Air
    4. Gempa : Richter
    5. Dosis : Obat
    Jawaban : D
    Suhu dapat diukur dengan skala celcius, sedangkan gempa dapat diukur dengan skala richter.
  3. SENDOK : MAKAN = _____ : _____
    1. Buku : Jual
    2. Sandal : Jepit
    3. Rambut : Sisir
    4. Gelas : Minum
    5. Siswa : Sekolah
    Jawaban : D
    Sendok digunakan untuk makan, sedangkan gelas digunakan minum.
  4. CINCIN : JARI = _____ : _____
    1. Bunga : Taman
    2. Meja : Kursi
    3. Kalung : Leher
    4. Kaki : Sepatu
    5. Obat : Sakit
    Jawaban : C
    Cincin dipakaikan di jari, sedangkan kalung dipakaikan di leher.
  5. MUDUN = ....
    1. Problema
    2. Beradab
    3. Referensi
    4. Setuju
    5. Mufakat
    Jawaban : B
    Mudun = beradab atau bertamadun.
  6. FUTURISTIS = ....
    1. Teknologi tinggi
    2. Canggih
    3. Nyaman
    4. Indah dan nyaman
    5. Menuju masa depan
    Jawaban : E
    Futuristis artinya terarah, tertuju ke masa depan.
  7. TANUR = ....
    1. Perapian
    2. Pucuk daun kelapa
    3. Tempat bersemedi
    4. Tempat memasak
    5. Perlambang
    Jawaban : A
    Tanur = perapian, dapur, tungku.
  8. TUA >< ....
    1. Perbani
    2. Baru
    3. Muncul
    4. Muda
    5. Nenek
    Jawaban : D
    Lawan kata yang tepat untuk kata tua adalah kata muda.
  9. CACAT >< ....
    1. Benar
    2. Baik
    3. Normal
    4. Tepat
    5. Jujur
    Jawaban : C
    Cacat = tidak normal. Lawan kata yang tepat untuk kata cacat adalah kata normal.
  10. INDIVIDUAL >< ....
    1. Soliter
    2. Kolektif
    3. Kolator
    4. Personel
    5. Selektif
    Jawaban : B
    Individual = perorangan. Lawan kata yang tepat untuk kata individual adalah kata kolektif.
