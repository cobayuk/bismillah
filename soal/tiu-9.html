Contoh Soal dan Pembahasan TIU SKD bagian 9

Jun 11, 2020

Jawablah pertanyaan-pertanyaan di bawah ini dengan baik dan benar! Kemudian cek jawabanmu dengan menekan tombol pembahasan yang ada di bawah setiap soal. Selamat mengerjakan!

  1. KEUNTUNGAN : PENJUALAN = ...
    1. Pendekar : silat
    2. Kemasyuran : keberanian
    3. Kebenaran : hukum
    4. Toleransi : kerukunan
    5. Pembelaan : jaksa
    Jawaban : B
    Keuntungan : penjualan = kemasyuran : keberanian. Keduanya sama - sama mempunyai hubungan tentang "diperoleh dari".
  2. RATA : MULUS = ...
    1. Bengkok : liku
    2. Kasar : aspal
    3. Manis : madu
    4. Obat : pahit
    5. Buah : segar
    Jawaban : A
    Rata : mulus = bengkok : liku. Keduanya sama - sama mempunyai hubungan tentang "akan terlihat".
  3. HEMPASAN : ABRASI = ...
    1. Lapar : Haus
    2. Reruntuhan : penghancuran
    3. Rayuan : pujian
    4. Memasak : panas
    5. Membakar : asap
    Jawaban : B
    Hempasan : abrasi = reruntuhan : penghancuran. Keduanya sama - sama mempunyai hubungan tentang "menyebabkan".
  4. KOMPOR : GAS = ...
    1. SPBU : mobil
    2. Sepeda motor : bensin
    3. Direktur : perusahaan
    4. Jaring : nelayan
    5. Gergaji : tukang kayu
    Jawaban : B
    Kompor : gas = sepeda motor : bensin. Keduanya sama - sama mempunyai hubungan "agar dapat digunakan membutuhkan".
  5. EMBERKASI = ....
    1. Penampungan
    2. Tempat pemberangkatan
    3. Gedung pertemuan
    4. Barak pengungsian
    5. Markas besar
    Jawaban : B
    Emberkasi artinya tempat pemberangkatan dengan pesawat terbang atau kapal laut.
  6. PURGATIF = ....
    1. Masuk akal
    2. Pembinaan
    3. Membersihkan
    4. Merambah
    5. Keterikatan
    Jawaban : C
    Purgatif = membersihkan, menyucikan, atau obat pencuci perut, pencahar.
  7. DERIVASI = ....
    1. Afiksal
    2. Kedatangan
    3. Muncul
    4. Pengurangan
    5. Kesalahan
    Jawaban : A
    Derivasi artinya pengimbuhan afiks yang tidak bersifat infeksi pada bentuk dasar untuk membentuk kata. Afiksasi artinya proses atau hasil penambahan afiksasi (prefiks, konfiks, sufiks) pada kata dasar.
  8. LABIL >< ....
    1. Hancur
    2. Rusuh
    3. Runtuh
    4. Retak
    5. Stabil
    Jawaban : E
    Labil = Tidak kokoh. Lawan kata yang tepat untuk kata labill adalah kata stabil.
  9. KOHESI >< ....
    1. Partisipasi
    2. Adhesi
    3. Agresi
    4. Asimilasi
    5. Regresi
    Jawaban : B
    Lawan kata yang tepat untuk kata kohesi adalah kata adhesi.
  10. PERTAHANAN >< ...
    1. Buka
    2. Serangan
    3. Perlindungan
    4. Tutup
    5. Maju
    Jawaban : B
    Lawan kata yang tepat untuk kata pertahanan adalah kata serangan.
