    1. Langsung mengeluarkan pendapat Anda juga walau ia pimpinan
    2. Menutup mulut karena tidak mau membantah pimpinan
    3. Mengiyakan saja untuk menyetujui pendapat pimpinan tersebut
    4. Meninggalkan ruangan rapat karena Anda geram dengan pimpinan Anda
    5. Biasa saja karena hal tersebut sudah sering terjadi
    Jawaban : A
    • Poin 5 : a
    • Poin 4 : e
    • Poin 3 : c
    • Poin 2 : b
    • Poin 1 : d
  3. Jika pasangan saya tidak menyukai tempat dimana saya bekerja maka sikap saya...

    3. Mempertimbangkan untuk berhenti dari pekerjaan saya
    4. Tetap memilih pekerjaan saya dengan resiko memutus hubungan dengan pasangan saya

  8. Anda menerima perintah tugas ke luar kota. Kebetulan pekerjaan anda di kantor menumpuk dan harus diselesaikan. Maka tindakan yang akan anda lakukan adalah...
    1. Meminta atasan untuk memberikan perpanjangan waktu menyelesaikan tugas tersebut
    2. Meminta atasan untuk keluar kota setelah anda menyelesaikan pekerjaan anda
    3. Mengkoordinasikan tugas yang dapat dipercayakan kepada rekan kerja, selain tidak menyalahi prosedur pekerjaan dapat diselesaikan tepat waktu
    4. Membawa pekerjaan tersebut ke luar kota. Sehingga dapat diselesaikan tepat waktu
    5. Menyelesaikan semampu Anda

    • Poin 5 : c

    • Poin 3 : a
    • Poin 2 : b
