    2. Membela diri
    3. Menolak tuduhan itu dan meminta penjelasan
    4. Menjelaskan duduk permasalahan yang sebenarnya
    5. Marah dan tidak terima

  2. Hari ini saya mendapat kesempatan mendapatkan beasiswa untuk pelajar muda internasional dan selama dua tahun studi saya tidak tentunya tidak diperkenankan menikah dulu. Sedangkan atas kesepakatan saya dan pihak keluarga calon istri, saya akan mempersiapkan pernikahan tahun ini. Keputusan saya adalah ...
    1. Menolak tawaran beasiswa tersebut
    2. Menerima tawaran beasiswa tersebut dengan konsekuensi saya harus menikah dulu karena sudah janji
    3. Menolak tawaran tersebut dan berharap bisa mendapat tawaran beasiswa lain setelah menikah
    4. Menerima tawaran beasiswa tersebut dan berat hati membatalkan pernikahan
    5. Mencari beasiswa lain setelah menikah saja

  4. Saya tahu maksud atasan saya meskipun beliau salah mengatakannya atau keliru saat memberi instruksi pada tim saya. Bagaimana sikap anda ...

    2. Meskipun dia atasan saya, saya menegurnya dengan keras atas kesalahan
    3. Saya mengulanginya meskipun dalam hati terasa sangat marah
    4. Saya tidak bersedia mengulanginya karena kesalahan bukan dari pihak saya
    5. Saya bersedia mengulanginya dan saya juga memintanya untuk lebih berhati - hati dalam memberikan instruksi di kemudian hari
